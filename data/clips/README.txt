MS Amma — pre-recorded human voice clips
========================================

Drop short audio files here to play a REAL, warm human voice instead of the synthetic
(robotic) voice for the fixed moments of class. Supported: .mp3, .wav, .m4a, .ogg.

Use these EXACT file names:

  greeting.mp3   — the opening welcome. Say it warmly, with a smile:
        "வணக்கம் கண்ணா! MS Amma இங்க இருக்கேன். இன்னைக்கு நம்ம class ஆரம்பிக்கலாமா?"

  closing.mp3    — the closing blessing. Say it warmly:
        "நீ சிறப்பாக பாடுவாய் கண்ணா! அடுத்த Sunday பார்க்கலாம். நன்றி."

How:
  • Record on any phone voice recorder in a quiet room.
  • Name the file exactly (greeting / closing) and put it in this folder
    (data/clips/). .wav or .mp3 both work.
  • If a clip is present, the app plays it; if not, it falls back to the
    synthetic voice automatically — so nothing breaks before you record.

Later we can add more clips (key Tamil music terms, "Superb kanna!", etc.) the same way.
